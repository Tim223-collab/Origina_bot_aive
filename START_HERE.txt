📋 ЧТО ДАЛЬШЕ? (5 минут)

═══════════════════════════════════════════════════════════════

┌────────────────────────────────────────────────────────────┐
│ ШАГ 1: УСТАНОВКА ЗАВИСИМОСТЕЙ                              │
└────────────────────────────────────────────────────────────┘

Windows:
  📌 Запусти: install.bat

Linux/Mac:
  📌 Запусти: chmod +x install.sh && ./install.sh


┌────────────────────────────────────────────────────────────┐
│ ШАГ 2: НАСТРОЙКА .env ФАЙЛА                                │
└────────────────────────────────────────────────────────────┘

После установки файл .env будет создан автоматически.
Открой его и заполни:

✅ TELEGRAM_BOT_TOKEN - получи у @BotFather
✅ ALLOWED_USER_IDS - получи у @userinfobot
✅ DEEPSEEK_API_KEY - получи на platform.deepseek.com
✅ WORK_SITE_URL - твой рабочий сайт (опционально)

📖 Подробнее: смотри QUICKSTART.md


┌────────────────────────────────────────────────────────────┐
│ ШАГ 3: ПРОВЕРКА НАСТРОЙКИ                                  │
└────────────────────────────────────────────────────────────┘

Windows:
  📌 Запусти: python test_config.py

Linux/Mac:
  📌 Запусти: python3 test_config.py

Это проверит что все настроено правильно.


┌────────────────────────────────────────────────────────────┐
│ ШАГ 4: ЗАПУСК БОТА                                         │
└────────────────────────────────────────────────────────────┘

Windows:
  📌 Запусти: start.bat

Linux/Mac:
  📌 Запусти: chmod +x start.sh && ./start.sh

Или вручную:
  📌 python main.py (Windows)
  📌 python3 main.py (Linux/Mac)


┌────────────────────────────────────────────────────────────┐
│ ШАГ 5: ПРОВЕРКА В TELEGRAM                                 │
└────────────────────────────────────────────────────────────┘

1. Открой Telegram
2. Найди своего бота
3. Напиши /start
4. Напиши "Привет" - бот должен ответить!

❓ ЧАСТЫЕ ВОПРОСЫ

═══════════════════════════════════════════════════════════════

Q: Бот не отвечает?
A: Проверь что:
   - Твой ID в ALLOWED_USER_IDS
   - Бот запущен и работает
   - Токен правильный

Q: Ошибка "playwright not found"?
A: Выполни: playwright install chromium

Q: Как остановить бота?
A: Нажми Ctrl+C в терминале

Q: Сколько стоит DeepSeek?
A: ~$0.14 за миллион токенов (очень дешево!)

Q: Можно запустить на сервере 24/7?
A: Да! Смотри DEPLOY.md
